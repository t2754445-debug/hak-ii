# Хак ИИ — Отряд №5

Сайт группы №5 «Хак ИИ» — школа бизнес-автоматизации, 39-й поток.

Шесть человек, один маскот-черепашка и одна цель — превратить ручную работу в автоматизированные процессы.

## Состав отряда

- **Владимир** — Лидер
- **Алина** — Психолог · транзактный анализ
- **Павел** — Маркетплейсы · товары для дома
- **Роман** — Сержант по ДЗ · сертификация кораблей
- **Светлана** — Мебель · креатив и нейросети
- **Ульяна** — Куратор · дизайн · «Кв Дорого»

## Структура

```
.
├── index.html       — главная страница
├── styles.css       — стили
├── script.js        — интерактив (лоадер, бургер, reveal-анимации)
└── media/           — фото команды, маскот, видео, аудио
```

## Запуск локально

```bash
python -m http.server 3030
```

Затем открой `http://localhost:3030/`.

## Деплой

Сайт опубликован через GitHub Pages из ветки `main`.
